# 난 농장 관리 시스템 MVP

난(오키드) 농장의 재배 로트, 작업, 출하 일정을 한 화면에서 관리하는 React 기반 MVP입니다.

## 주요 기능

- 대시보드 핵심 지표: 총 재배 수량, 평균 건강도, 주의 로트, 출하 가능 수량
- 재배 로트 현황: 품종/온실/생육 단계 검색, 온습도와 건강도 확인
- 신규 로트 등록: 품종, 온실, 생육 단계, 수량 입력
- 작업 관리: 오늘의 작업 목록과 완료 처리
- 출하 관리: 거래처별 예정 출하 일정 확인

## 실행 방법

```bash
npm install
npm run dev
```

## 빌드

```bash
npm run build
```
