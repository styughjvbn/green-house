# 분석 페이지 기능 명세서

## 1. 기능 설명

분석 페이지는 농장 운영 데이터를 기반으로 **판매 흐름, 농장 상태, 작업 이력**을 확인하고 의사결정을 돕는 화면이다.

---

## 2. 핵심 역할

```text
1. 매출 및 출하 흐름 확인
2. 품종별/거래처별 성과 확인
3. 농장 공간 사용 현황 확인
4. 작업 이력 및 상태 이상 확인
5. 판매 가능 수량 파악
```

분석 결과에서 관련 화면으로 이동할 수 있어야 한다.

---

## 3. 화면 구성

```text
[매출/출하] [품종 분석] [거래처 분석] [농장 공간] [작업/상태]
```

상단에는 요약 카드와 공통 필터를 제공한다.

---

## 4. 상단 요약 카드

### 표시 정보

```text
이번 달 매출
출하 수량
미입금 금액
판매 가능 수량
상태 이상 수
분갈이 예정 수
```

### 동작

카드 클릭 시 관련 탭 또는 화면으로 이동

---

## 5. 공통 필터

```text
기간
동 / 배드 / 구역
품종
거래처
상태 / 판매 상태 / 입금 상태
```

* 필터 변경 시 현재 탭 데이터 갱신
* 기간 필터는 전체 탭 공통 적용

---

## 6. 매출/출하 탭

### 주요 기능

```text
매출 추이 확인
출하 수량 확인
미입금 전표 확인
최근 판매 전표 확인
```

### 주요 동작

```text
전표 클릭 → 판매 관리 이동
품종 클릭 → 품종 분석 이동
거래처 클릭 → 거래처 분석 이동
```

---

## 7. 품종 분석 탭

### 주요 기능

```text
품종별 보유 수량
판매 수량 및 매출
상태 이상 및 분갈이 예정
```

### 주요 동작

```text
품종 선택 → 상세 정보 확인
난 묶음 보기 → 농장 현황 이동
판매 전표 보기 → 판매 관리 이동
```

---

## 8. 거래처 분석 탭

### 주요 기능

```text
거래처별 매출
미입금 금액
거래 빈도
주요 구매 품종
```

### 주요 동작

```text
거래처 클릭 → 전표 목록 이동
미입금 클릭 → 미입금 전표 필터 적용
```

---

## 9. 농장 공간 탭

### 주요 기능

```text
동/배드/구역 사용률
상태 이상 분포
빈 구역 확인
```

### 주요 동작

```text
동/배드 클릭 → 농장 현황 이동
빈 구역 클릭 → 난 묶음 관리 이동
```

---

## 10. 작업/상태 탭

### 주요 기능

```text
작업 이력 확인
누락 작업 확인
상태 이상 추이 확인
```

### 주요 동작

```text
누락 구역 클릭 → 농장 현황 이동
상태 이상 클릭 → 난 묶음 관리 이동
작업 추가 → 작업 이력 이동
```

---

## 11. 출하 가능 수량

```text
판매 가능 난 묶음 수
품종별/동별 판매 가능 수량
```
