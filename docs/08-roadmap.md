# 향후 확장 계획

## 1. 우선순위 높음

### 판매·경매·입금 안정화

- 경매 lot과 정산 묶음 연결 강화
- 부분입금 처리 UX 개선
- 입금 자동 매칭 후보 관리
- 은행 입금 내역 가져오기
- 정산 재계산 이력 보존

### 작업 이력 보정

- 작업 이력 수정/취소 정책 정리
- 잘못 입력한 작업에 대한 보정 흐름
- 작업 유형 관리 UI 개선

### 데이터 가져오기

- 엑셀/CSV 가져오기
- 원본 행 보존
- 검수 상태 관리
- 중복/누락 탐지

## 2. 중간 우선순위

### 재고 연동

- 판매 품목과 난 묶음 연결
- 출고 완료 시 수량 차감
- 전표 취소 시 수량 복구
- 재고 변동 이력

### 배치 정밀화

- 논리 구역 병합/분할
- 세그먼트별 수용량 관리
- 15구/20구/24구 판 단위 관리
- 행잉 구역 관리
- 압축/임시 배치 경고

### 분석 화면

- 품종별 매출
- 거래처별 매출/미입금
- 경매장별 유찰률/반환률
- 동/배드/구역 사용률
- 작업 누락 구역

## 3. 장기 확장

### 사진·QR

- 난 묶음 사진 기록
- 개별 화분 QR 관리
- 고가품/모주 개체 관리
- 사진과 작업 이력 연결

### 모바일 현장 입력

- 모바일 전용 UI
- QR 스캔
- 음성 메모
- 오프라인 임시 저장

### IoT·AI

- 온도/습도/광량/CO₂ 센서 연동
- 병해충 의심 사진 분류
- 작업 추천
- 출하 가능 수량 예측

### 외부 연동

- 회계 프로그램 연동
- 택배/송장 연동
- 스마트스토어/블로그/인스타그램 판매 채널 연동

## 4. 보류 또는 신중 검토

다음 기능은 바로 구현하지 않는다.

- 정밀 CAD 수준의 좌표 편집
- 개별 화분 전수 관리
- 완전 자동 재고 차감
- 자동 입금 확정 처리
- AI 진단을 운영 판단의 단독 근거로 사용하는 기능
