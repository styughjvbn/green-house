# 기능 요약

## 1. 대시보드

농장 운영 상태를 빠르게 확인하는 진입 화면이다.

표시 대상:

- 전체 동/배드/구역 수
- 전체 난 묶음 수
- 상태 이상 수
- 분갈이 예정 수
- 최근 작업
- 판매/입금 요약
- 빠른 이동 링크

대시보드는 상세 수정 화면이 아니다.

## 2. 농장 현황

전체 농장을 조회하는 화면이다.

핵심 역할:

```text
전체 농장 보기
→ 동 선택
→ 물리 배드 선택
→ 논리 구역 선택
→ 선택 범위의 난 묶음 목록 확인
```

농장 현황에서는 데이터 변경을 최소화한다.

하지 않는 작업:

- 난 묶음 생성
- 난 묶음 수정
- 난 묶음 삭제
- 드래그 이동

수정이 필요하면 난 묶음 관리 화면으로 이동한다.

## 3. 난 묶음 관리

난 묶음을 실제로 생성, 수정, 삭제, 이동하는 작업 화면이다.

주요 기능:

- 동 선택
- 선택 동 상세맵 표시
- 물리 배드 1/2/3 좌→우 표시
- 배드별 좌/우 논리 구역 표시
- 난 묶음 등록/수정/삭제
- 같은 동 내부 드래그 이동
- 다른 동 이동 목적지 선택
- 이동 후 위치 이동 이력 자동 생성
- 배치 추천/자리 차지 정보 확인

## 4. 작업 이력

농약, 비료, 분갈이, 정리, 상태 기록, 메모, 위치 이동을 기록한다.

주요 기능:

- 작업 이력 등록
- 대상 유형 선택
- 기간/대상/작업 유형 필터
- 작업 유형 추가/수정/비활성화/정렬
- 위치 이동 이력 자동 생성

운영 원칙:

- 작업 이력은 가능하면 삭제하지 않는다.
- 잘못 입력한 경우 수정 또는 보정 이력을 남기는 방향을 우선한다.

## 5. 판매 관리

거래처와 판매 전표를 관리한다.

주요 기능:

- 거래처 등록/조회
- 판매 전표 생성
- 판매 품목 입력
- 품목 금액과 전표 총액 자동 계산
- 판매 전표 목록/상세 조회
- A5 전표 출력
- 입금 확인

MVP 기준에서는 전표 생성만으로 난 묶음 수량을 자동 차감하지 않는다.

## 6. 경매 추적

경매장 출하 물량을 lot 단위로 추적한다.

주요 기능:

- 경매 lot 조회
- 경매 상태 변경
- 유찰/재경매 대기/반환 확인
- 수량 보정
- 경매 결과 행 관리
- 경매장별 정산 생성
- 정산 재계산

핵심은 행 합계가 아니라 **출하 lot별 상태 추적**이다.

## 7. 입금·정산

거래처별 돈 흐름을 관리한다.

주요 기능:

- 판매 전표 입금 확인
- 경매 정산 입금 확인
- 부분입금 처리
- 거래처 잔액 요약
- 입금 이벤트 목록 조회
- 거래처별 정산 설정

## 8. 품종/입고/자재 관리

품종 기준 정보와 입고 흐름, 자재 기준 정보를 함께 관리한다.

주요 기능:

- 품종 등록/수정/비활성화
- 품종별 연결 난 묶음 조회
- 입고 등록
- 입고 상세 수정
- 유리병 모종 포트 작업 등록
- 입고 취소
- 자재 등록/수정/비활성화

운영 원칙:

- 품종과 자재는 삭제보다 비활성화를 우선한다.
- 입고 기록은 난 묶음 생성 전까지만 취소 가능하다.
- 유리병 모종과 즉시 배치 입고는 입력 필드와 후속 흐름이 다르다.

## 9. 분석

농장 운영 데이터를 바탕으로 판단을 돕는 화면이다.

분석 후보:

- 매출/출하 추이
- 품종별 판매 성과
- 거래처별 매출/미입금
- 동/배드/구역 사용률
- 작업 누락 구역
- 상태 이상 추이
- 경매장별 유찰률/반환률/평균 단가
