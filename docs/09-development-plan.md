## 2. MVP 완료 기준

다음이 가능하면 MVP 완료로 본다.

- 15개 동, 45개 물리 배드, 90개 기본 논리 구역이 생성된다.
- 물리 배드가 실제 농장 기준 좌→우 순서로 정렬된다.
- 농장 현황 화면에서 전체 농장맵이 제공된다.
- 전체 농장맵에서 동을 클릭하면 해당 동의 모든 난 묶음 목록이 표시된다.
- 줌인 후 물리 배드를 클릭하면 해당 배드의 모든 난 묶음 목록이 표시된다.
- 논리 구역을 클릭하면 해당 구역의 난 묶음 목록과 최근 작업 요약이 표시된다.
- 난 묶음 관리 화면에서 선택한 동 상세맵이 제공된다.
- 동 상세맵에서 3개 물리 배드가 좌→우로 표시된다.
- 난 묶음을 등록/수정/삭제할 수 있다.
- 난 묶음 블록이 난 묶음 관리 화면에 표시된다.
- 같은 동 내부에서 난 묶음을 드래그 앤 드롭으로 이동할 수 있다.
- 다른 동 이동은 목적 동/배드/논리 구역 선택 방식으로 처리할 수 있다.
- 난 묶음 이동 시 위치 이동 작업 이력이 자동 생성된다.
- 농약/비료/분갈이/잎 정리/잡초 정리/단화 작업 이력을 등록할 수 있다.
- 구역 또는 배드별 최근 작업일이 표시된다.
- 판매 전표를 등록할 수 있다.
- 판매 품목의 금액과 총액이 자동 계산된다.
- 판매 전표를 A5 기준으로 출력할 수 있다.
- 논리 구역 또는 배드별 관리표를 A5 기준으로 출력할 수 있다.
